Frequently Asked Questions

What is Morse code?
Morse code is a communication system using dots (short signals) and dashes (long signals) to represent letters and numbers. It was invented by Samuel Morse in the 1830s and used extensively for telegrams and maritime communication.
How do you write SOS in Morse code?
SOS is written as ... --- ... (three dots, three dashes, three dots). It was chosen as a distress signal because it is simple, distinctive, and easy to transmit even under stress.
What is a dit and a dah?
A dit (dot) is a short signal lasting one unit of time. A dah (dash) is a long signal lasting three units. The gap between symbols within a letter is one unit, between letters is three units, and between words is seven units.
Is Morse code still used today?
Yes. Morse code is still used by amateur radio operators worldwide, in aviation (airport beacons broadcast their identifier in Morse), and it remains a skill taught in military training and survival courses.
